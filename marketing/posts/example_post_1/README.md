# Post: Gestión de Cancelaciones y Recuperos

## Descripción

Este post muestra el problema de gestionar cancelaciones y recuperos manualmente a través de WhatsApp, y cómo TuTaller automatiza todo el proceso.

## Narrativa Visual

1. **Visual 1**: Mensaje de WhatsApp un domingo a la noche preguntando sobre horarios
2. **Visual 2**: Otro mensaje pidiendo reprogramar una clase por complicaciones en el trabajo
3. **Visual 3**: Dos chats simultáneos - uno cancela, otro quiere entrar - el caos del medio
4. **Visual 4**: La solución: TuTaller lo hace automáticamente (cancela, devuelve crédito, avisa al siguiente)
5. **Visual 5**: Beneficio final: los alumnos reservan en 3 clicks, vos te ocupás de enseñar
