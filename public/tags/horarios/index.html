Tutorial 3: Crear y Gestionar Clases Recurrentes

Las clases recurrentes son la base de tu programación regular. En este tutorial vas a aprender a crear clases que se repiten de forma automática (por ejemplo, todos los lunes a las 18:00), ahorrándote el trabajo de crear cada clase individual manualmente.

¿Qué son las Clases Recurrentes?

Una clase recurrente es aquella que se repite en un patrón regular:

  • Ejemplo 1: Yoga todos los lunes, miércoles y viernes a las 19:00
  • Ejemplo 2: Spinning todos los martes y jueves a las 7:00
  • Ejemplo 3: Taller de Cerámica todos los sábados a las 10:00

Una vez configuradas, estas clases se generan automáticamente y los alumnos pueden reservar sus turnos.
