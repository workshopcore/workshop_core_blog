Tutorial 11: Reservas desde la Perspectiva del Alumno

Tutorial 11: Reservas desde la Perspectiva del Alumno

La función principal de la app para tus alumnos es reservar clases. El proceso está diseñado para ser súper simple e intuitivo.

La Sección “Reservar”

Desde el menú principal, el alumno accede a “Reservar”. Acá ve un listado o calendario con todas las clases disponibles para los próximos días.

El Proceso de Reserva

  1. El alumno elige la clase que quiere (ej: “Yoga - Lunes 19hs”).
  2. Hace clic en “Reservar”.
  3. El sistema verifica si tiene créditos suficientes.
    • Si tiene créditos: La reserva se confirma inmediatamente y se descuenta 1 crédito.
    • Si no tiene créditos: El sistema le avisa que debe recargar o contactar al administrador.

Mis Clases

Una vez reservada, la clase aparece en la sección “Mis Clases” del alumno. Desde ahí puede ver el detalle o cancelar la reserva si no puede asistir (lo que te libera el cupo automáticamente).

Tutorial 14: Gestionar Notificaciones (Alumno)

Tutorial 14: Gestionar Notificaciones

La comunicación fluida es clave. En TuTaller, los alumnos reciben notificaciones importantes sobre:

  • Confirmaciones de reserva.
  • Cancelaciones de clases.
  • Recordatorios de vencimiento de créditos.
  • Mensajes directos tuyos (como vimos en el Tutorial 8).

El Centro de Notificaciones

Los alumnos pueden acceder a sus mensajes haciendo clic en el ícono de la campana o en la sección “Notificaciones” del menú.

Funcionalidades:

  • Leer: Al hacer clic en una notificación, se expande para mostrar el mensaje completo.
  • Marcar como leída: Una vez leída, pueden archivarla para mantener su bandeja limpia.
  • Estado: Las notificaciones no leídas suelen destacarse para llamar la atención.

Enseñarles a tus alumnos a revisar esta sección te asegura que no se pierdan ninguna novedad importante de tu espacio.

Tutorial 13: Suscripciones a Clases Recurrentes

Tutorial 13: Suscripciones a Clases Recurrentes

Para los alumnos que asisten siempre en el mismo horario (por ejemplo, “Yoga los Martes 19hs”), TuTaller ofrece la función de Suscripciones. Esto les permite asegurar su lugar automáticamente cada semana sin tener que reservar manualmente clase por clase.

¿Cómo Suscribirse?

Desde la sección “Suscripciones” (o “Clases Semanales”) en el menú, el alumno puede ver todas las clases recurrentes disponibles.

Pasos para el Alumno:

  1. Busca la clase deseada en la lista.
  2. Hace clic en “Suscribirse”.
  3. El sistema verificará si tiene créditos suficientes.
  4. ¡Listo! Queda anotado fijo para esa clase todas las semanas.

Ventajas

  • Comodidad: Se olvidan de reservar cada semana.
  • Seguridad: Su lugar está guardado, nadie se lo puede sacar.
  • Organización: Vos como administrador ya sabés con qué base de alumnos contás.

Continuá con el Tutorial 14: Gestionar Notificaciones para aprender a mantenerte al día con las novedades.
