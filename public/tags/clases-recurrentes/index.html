Tutorial 13: Suscripciones a Clases Recurrentes

Para los alumnos que asisten siempre en el mismo horario (por ejemplo, “Yoga los Martes 19hs”), TuTaller ofrece la función de Suscripciones. Esto les permite asegurar su lugar automáticamente cada semana sin tener que reservar manualmente clase por clase.

¿Cómo Suscribirse?

Desde la sección “Suscripciones” (o “Clases Semanales”) en el menú, el alumno puede ver todas las clases recurrentes disponibles.

Pasos para el Alumno:

  1. Busca la clase deseada en la lista.
  2. Hace clic en “Suscribirse”.
  3. El sistema verificará si tiene créditos suficientes.
  4. ¡Listo! Queda anotado fijo para esa clase todas las semanas.

Ventajas

  • Comodidad: Se olvidan de reservar cada semana.
  • Seguridad: Su lugar está guardado, nadie se lo puede sacar.
  • Organización: Vos como administrador ya sabés con qué base de alumnos contás.
