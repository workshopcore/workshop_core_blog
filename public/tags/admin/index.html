Tutorial 1: Introducción a TuTaller - Panel de Administración

¡Bienvenido a TuTaller! En este primer tutorial, vamos a recorrer el panel de administración y conocer todas las herramientas que tenés disponibles para gestionar tu espacio de clases de forma eficiente.

¿Qué es TuTaller?

TuTaller es una plataforma completa diseñada para facilitar la gestión de espacios de clases, talleres y actividades. Te permite administrar turnos, usuarios, créditos, enviar notificaciones y mucho más, todo desde una interfaz intuitiva y fácil de usar.
